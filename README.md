# Absolute Value Calculator (C++)

## 📌 Description
This program calculates the absolute value of a number.

## 🧠 Logic Used
- if condition

## ⚙️ How it Works
- User enters a number
- If number is negative → it is converted into positive
- If number is already positive → it remains the same

## 💻 Example
Input: -5  
Output: 5  

Input: 10  
Output: 10  

## 🚀 Concepts Practiced
- Conditional statements (if)
- Basic input/output
